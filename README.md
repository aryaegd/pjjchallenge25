# PJJ Challenge Lucky Draw

Lucky draw system untuk kompetisi PJJ Challenge dengan 3 kategori:
- Brain Challenge
- Essay
- Poster

## Features
- Upload file Excel/CSV atau input manual
- Random selection 5 pemenang per kategori
- Responsive design dengan animasi
- Export hasil ke CSV
- Riwayat undian tersimpan di browser

## Tech Stack
- HTML5, CSS3, JavaScript (Vanilla)
- XLSX.js untuk parsing file Excel
- Font Awesome untuk ikon
- Google Fonts (Poppins)

## Deployment di Vercel
1. Push ke repository GitHub
2. Connect repository ke Vercel
3. Deploy otomatis

## Local Development
Buka `index.html` di browser atau gunakan live server.
